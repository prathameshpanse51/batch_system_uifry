# batch-system-uifry

 ## Installation
1. Clone the repository:
    ```sh
     https://github.com/prathameshpanse51/batch_system_uifry.git
    ```
2. Install dependencies:
    ```sh
      npm install
    ```
3. Run the project locally to see all features in action:
    ```
      npm run dev
    ```
